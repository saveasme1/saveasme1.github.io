앱 설치

기기에 맞는 설치 방법만 보여 드립니다.

감지 중…

아이폰 제일 쉬운 방법 (QR)

  1. 아이폰 기본 카메라로 아래 QR을 찍습니다.
  2. 위에 뜨는 알림을 눌러 Safari로 엽니다.
  3. Safari 하단 공유 → 홈 화면에 추가

카메라 QR은 Safari로 바로 열려서 카톡/네이버보다 훨씬 쉽습니다. (공유 단계는 Apple 정책상 생략 불가)

설치 없이 바로 열기

이미 Safari에 있다면

  1. 하단 공유 버튼
  2. 홈 화면에 추가 → 추가

주소 복붙은 필요 없습니다. 공유 → 홈 화면에 추가만 하면 됩니다.

Android

  1. Chrome 또는 삼성 인터넷으로 이 페이지를 엽니다.
  2. 위 앱 설치하기를 누르거나
  3. 브라우저 메뉴(≡) → 홈 화면에 추가

카톡 안이면 위 “브라우저에서 열기”로 Chrome/인터넷 앱을 먼저 여세요.

PC / 기타

  1. 휴대폰에서 이 링크를 여세요.
  2. 휴대폰 브라우저에서 앱 설치하기를 사용하세요.
